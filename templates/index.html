About

This website aims to be a public archive of weather data of all the cities around the world.

Currently, the supported cities are:

  • Murree, Pakistan. Data from 2006-2012.

Query database

The following options are available:

  1. For a given year display the highest temperature and day, lowest temperature and day, most humid day and humidity.
    • Endpoint: /task1
    • Required params: (1) y -> year to query
    • For example: /task1?y=2010
  2. For a given month display the average highest temperature, average lowest temperature, average mean humidity.
    • Endpoint: /task2
    • Required params: (1) y -> year to query (2) m -> month to query
    • For example: /task2?y=2010&m=2
  3. For a given month draw two horizontal bar charts on the console for the highest and lowest temperature on each day.
    • Endpoint: /task3
    • Required params: (1) y -> year to query (2) m -> month to query
    • For example: /task3?y=2010&m=2
  4. Multiple reports - Generate reports of more than one task.
    • Endpoint: /multiple_reports
    • Params: Params of respective tasks will be the same. An additional taskX=true will be passed if we want to generate report of a specific task.
    • Params will be named as yX and mX, where X -> task number
    • For example: Params for task1 will be 'y1'. 'y2' & 'm2' for task2, and so on.
    • For example: /multiple_reports?task1=true&y1=2011&task2=true&y2=2010&m2=2
